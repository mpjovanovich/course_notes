## Example 1

"All dogs are mammals."

- Subject: dogs
- Predicate: mammals
- Quantifier: all

Let D: x is a dog
Let M: x is a mammal
Then: ∀x(D(x) → M(x))

"For all x, if x is a dog, then x is a mammal."
